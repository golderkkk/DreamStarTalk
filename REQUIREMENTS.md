# AI Chat Studio — 需求文档

> 版本：v2.0 | 更新日期：2026-07-05 | 技术栈：Flutter 3.24 + Dart 3.5 + Riverpod + Hive + GoRouter

---

## 一、产品概述

AI Chat Studio 是一款 AI 角色扮演聊天应用，支持用户创建角色卡、构建世界观设定，并与 AI 进行沉浸式角色扮演对话。核心设计参考 SillyTavern 的上下文管理策略，提供专业级对话风格控制。

**核心能力：** 角色创建 → 世界观构建 → AI 对话 → 风格微调 → 语音朗读

---

## 三、模块详细需求

### 3.1 对话模块（Chat）

#### 3.1.1 对话列表页

| 功能 | 说明 |
|------|------|
| 列表展示 | 按最后消息时间倒序，显示角色头像、标题、最后消息预览、时间戳、消息数 |
| 搜索 | 实时搜索对话标题、角色名、消息内容 |
| 快速开始 | FAB 弹出角色选择面板，选择后直接创建对话（跳过向导） |
| 完整向导 | 通过"完整向导"入口进入 3 步创建流程 |
| 左滑删除 | Dismissible + 确认弹窗 + 3 秒撤销 SnackBar |
| 长按菜单 | 继续对话、查看世界观信息、删除 |
| 骨架屏 | 8 个 shimmer 占位卡片 |
| 交错入场 | 每项延迟 60ms 渐入动画 |

#### 3.1.2 创建对话向导（3 步）

**Step 1 — 选择角色：**
- 展示所有角色列表，支持搜索
- 点击选择（高亮 + 勾选图标）
- 无角色时显示"去创建角色"按钮

**Step 2 — 选择世界观：**
- "跳过，不使用世界观"选项（默认选中）
- 展示所有世界观列表，支持搜索
- 可选一个世界观关联

**Step 3 — 主角设置：**
- 必填：你的称呼（对话中 AI 如何称呼你）
- 选填：简介、性格、外貌、背景故事
- 自定义字段：可添加任意 key-value 属性
- NSFW 模式开关
- 空称呼提交时显示错误提示

**创建后：** 保留返回栈（push），可返回对话列表

#### 3.1.3 聊天页面

**消息展示：**
- 用户消息：右侧，紫色渐变气泡
- AI 消息：左侧，深色卡片气泡 + 角色头像
- 系统消息：居中，半透明背景
- 错误消息：居中，红色边框
- 图片消息：250×250 缩略图，点击全屏预览（支持缩放）
- 日期分隔线：跨天自动显示
- 流式输入指示器：脉冲圆点 + "正在生成..." + 停止按钮

**消息操作：**
- TTS 朗读：每条 AI 消息下方的音量图标
- 消息搜索：顶部搜索图标，N/M 匹配计数，上下导航

**对话风格设置：**
- 入口：AppBar 风格按钮（可见）+ 长按角色头像（快捷）
- 面板内容（40+ 可调参数）：
  - 活人感：人格补充、情感基准、生动化
  - 情感基调：治愈/伤感/积极/消极
  - 字数设定：启用后可设最小/最大字数
  - 特化模式：恋爱文、NSFW、反差
  - 附加选项：防复述、增加对白、外表美化、深度写作
  - 人称视角：第一/二/三人称、非 user 视角、群像

**空对话状态：**
- 角色头像 + 名称
- 上下文感知的快捷回复建议
- 对话风格设置提示

**聊天输入：**
- 文本输入框（回车换行）
- 附件按钮：图片选择、拍照、文件
- @mention：输入 @ 触发角色选择器（多角色模式）
- 发送/停止按钮切换

#### 3.1.4 多角色对话

- 对话可关联多个角色（主角色 + 附加角色）
- @mention 指定消息目标角色
- 对话风格面板中可切换活跃角色

### 3.2 角色模块（Character）

#### 3.2.1 角色列表页

| 功能 | 说明 |
|------|------|
| 双列网格 | 渐变卡片展示头像、名称、简介、标签 |

| 长按菜单 | 编辑、开始对话（快速创建）、复制、导出角色卡、删除（撤销） |

| 模板创建 | 内置预设角色模板 |
| 骨架屏 | 6 个 shimmer 占位卡片 |

#### 3.2.2 角色编辑页

| 字段 | 类型 | 说明 |
|------|------|------|
| 名称 | 必填 | 角色名称 |

| 说话风格 | 文本 | 口头禅、语气特点 |
| 背景故事 | 多行 | 角色的详细背景 |

| 标签 | 标签 | 逗号分隔，用于分类筛选 |

### 3.3 世界观模块（World）

#### 3.3.1 世界观列表页

- 封面图卡片 + 渐变遮罩
- 搜索、长按菜单（编辑、开始对话、删除）
- FAB 创建新世界观

#### 3.3.2 世界观编辑页（4 Tab 布局）

**Tab 1 — 基本信息：**
- 名称（必填）、描述、世界规则、历史背景、封面图

**Tab 2 — 场景管理：**
- 场景列表：名称、描述、图片、时间、天气、氛围
- 可设置当前活跃场景（注入对话上下文）
- 关联 NPC

**Tab 3 — NPC 管理：**
- NPC 属性：名称、头像、描述、性格、背景、说话风格、关系、标签
- 从 NPC 库导入（跨世界复用）
- 关系图谱标签

**Tab 4 — 剧情线：**
- 剧情线类型：主线/支线/个人/事件
- 状态：进行中/暂停/完成/失败
- 剧情节点：对话/事件/选择/战斗/发现
- 分支选择：条件判断 + 效果触发
- 节点链式导航

#### 3.3.3 NPC 库

- 独立于世界的共享 NPC 池
- 支持搜索、新增、编辑
- 选择模式：批量导入到指定世界观

### 3.4 AI 提供商模块

#### 3.4.1 支持的提供商

| 提供商 | 类型 | 默认端点 |
|--------|------|----------|
| MiMo | mimo | https://api.xiaomimimo.com/v1 |
| DeepSeek | deepseek | https://api.deepseek.com |
| OpenAI | openai | https://api.openai.com |
| Claude | claude | https://api.anthropic.com |
| Gemini | gemini | https://generativelanguage.googleapis.com |
| Ollama | ollama | http://localhost:11434 |
| 统一网关 | gateway | 自定义 |

#### 3.4.2 提供商配置

- API Key（必填）
- 自定义端点（可选）
- 默认模型选择
- 启用/禁用开关

#### 3.4.3 模型配置

按功能分类配置不同模型：
- **文字对话** — 日常对话和文本生成
- **图片识别** — 分析和理解图片内容
- **语音合成** — 文本转语音
- **代码生成** — 编写和调试代码

每个功能可指定独立的提供商 + 模型，或使用默认配置。

## 四、上下文管理策略

参考 SillyTavern 的上下文构建流程：

```
┌──────────────────────────────────────┐
│ 1. 系统提示词（角色卡 + 世界观 + 主角） │
├──────────────────────────────────────┤
│ 2. 对话风格指令（40+ 可调参数）         │
├──────────────────────────────────────┤
│ 3. 自定义系统提示词（如有）             │
├──────────────────────────────────────┤
│ 4. 历史消息（智能截断，保留最近消息）    │
│    user → assistant → user → ...     │
├──────────────────────────────────────┤
│ 5. 最新用户消息                        │
└──────────────────────────────────────┘
```

**Token 预算分配：**
- 系统提示：动态计算
- 回复空间：预留 800 tokens
- 历史消息：剩余预算内从最新向前填充
- 首条消息：新对话时自动注入角色 greeting

**多模态支持：**
图片消息自动转为 base64，以 OpenAI vision API 格式发送。

## 七、已知限制

- 文件附件仅发送文件名文本，不传输实际文件内容
- 对话标题自动生成（前 20 字符），不支持手动编辑
- 世界观预设模板数量有限（5 种）
- 角色预设模板数量有限（10 种）
- 无云端同步功能
- 无对话导出功能（仅角色可导出）
