# AI Chat Studio — 产品报告文档

> **版本**: v2.0 | **更新日期**: 2026-07-06 | **技术栈**: Flutter 3.24 + Dart 3.5 + Riverpod + Hive + GoRouter
> **代码规模**: 71 个 Dart 文件，约 16,000 行代码

---

## 一、产品概述

AI Chat Studio 是一款现代化的 AI 角色扮演聊天应用，支持用户创建角色卡、构建世界观设定，并与 AI 进行沉浸式角色扮演对话。应用采用深色主题设计，提供专业级对话风格控制和多角色对话能力。

**核心价值主张**: 让用户与 AI 角色进行自然、沉浸式的对话体验，支持丰富的角色定制和世界观构建。

## 四、对话风格系统

参考 SillyTavern 的预设系统，提供 40+ 可调参数：

| 类别 | 选项 |
|------|------|
| 活人感 | 人格补充、情感基准、生动化、锋芒未露 |
| 情感基调 | 治愈、伤感、积极、消极 |
| 字数设定 | 启用/禁用，最小/最大字数 |
| 特化模式 | 恋爱文、NSFW、反差 |
| 附加选项 | 防复述、增加对白、外表美化、深度写作 |
| 人称视角 | 第一/二/三人称、非 user 视角、群像 |
| 补丁模块 | 抗过拟合、抗绝望、抗抢话、人物塑造、详略得当等 |

风格指令直接合并到系统提示词的 `[Writing Directives]` 段落，确保 AI 模型遵循。

## 五、上下文管理策略

参考 SillyTavern 的上下文构建流程：

```
[System Instruction]     ← 角色扮演核心指令（11 条规则）
[Character Card]         ← 角色卡信息
[World Setting]          ← 世界观设定
[NPCs]                   ← NPC 信息
[User Character]         ← 主角信息
[Writing Directives]     ← 对话风格指令
[Content Policy]         ← NSFW 指令（如启用）
---
历史消息（智能截断，保留最近消息）
用户最新消息
```

**Token 预算分配：**
- 系统提示：动态计算
- 回复空间：预留 800 tokens
- 历史消息：剩余预算内从最新向前填充

## 十一、操作流程

### 创建对话流程
```
对话列表 → FAB → 选择角色（可多选）→ 选择世界观（可跳过）→ 设置主角 → 开始对话
```

### 快速对话流程
```
角色列表 → 长按角色 → 开始对话 → 直接进入聊天
```

### 多角色对话流程
```
创建对话 → 选择 2+ 角色 → 开启多角色模式 → 发送消息 → 所有角色轮流回复
```

### @mention 流程
```
多角色对话 → 长按角色头像 → 选择目标 → 输入消息 → 发送 → 仅目标角色回复
```

### 对话风格调整流程
```
聊天页 → 点击风格按钮 → 开启选项 → 实时生效
```

### TTS 语音流程
```
设置 → AI 服务 → TTS 配置 → 输入 API Key → 选择音色 → 保存
聊天页 → 点击消息音量图标 → 语音朗读
```
